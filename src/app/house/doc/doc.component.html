Локальное подключение по Wi-Fi
При отсутствии связи с устройством, возможно подключится к нему локально по WI-FI. Супервайзер может получить Логин и пароль к WI-FI зайдя в Web-интерфейс во вкладке Элементы в поле Сеть. Техник(промывщик) и Mерчендайзер(сменщик кег) должен получить пароль и логин у Cупервайзера(руководства).
Выберите в настройках Сеть и Интернет Выберите Wi-Fi Нажмите добавить сеть Введите имя сети Выберите защиту сети WPA/WPA2 PSK Если в расширенных настройках есть настройка скрытая сеть, то выберите да. Введите пароль сети и добавьте сеть Выберите добавленную точку доступа для авторизации в сети Нажмите на название точки доступа, чтобы войти сеть.
